### Phone 端

安卓端代码，有以下特性支持：

1. UI设计。
2. 支持输入查询。
3. 支持 socket 通信——局域网内假设的服务器（任何支持 socket 的设备）进行信息交互。
4. 支持结果返回。





------



缓存多了，可以删除的目录有：

1. `./gradle`

2. `%USERPROFILE%\.gradle（普遍超过8GB）`



卸载 `Android Studio` 的时候可以删除这几个：

1. `%Appdata%\..\Local\Google`

2. `%Appdata%\..\Local\JetBrains\Toolbox\apps\AndroidStudio`
